# Validador de CPF

Este repositório contém uma aplicação em Python projetada para ler um Cadastro de Pessoas Físicas (CPF) e realizar sua validação matemática completa, determinando se ele é autêntico ou não.

## 🎯 Objetivo do Projeto
O objetivo principal deste projeto é aplicar lógica de programação e operações matemáticas para reproduzir o algoritmo oficial da Receita Federal na validação de CPFs. O sistema verifica a integridade dos dígitos verificadores e trata exceções de falsos-positivos (como números repetidos)
